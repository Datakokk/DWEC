Entuerto

1. Explicar qué pasa y por qué en el estado actual:

            function trocearArrayEnTrozos(array, tamTrozo) {
                const nuevoArray = [];
              
                let j = 0, cont = 0;
                for(let i = 0; i < array.length; i++) {
                  var nuevoNuevoArray = [];
                  if(j < tamTrozo) {
                    nuevoNuevoArray[j++] = array[i];
                  } else {
                    j = 0;
                    nuevoArray[cont++] = nuevoNuevoArray;
                  }
                }
              
                return nuevoArray;
              }

2. Arreglar el entuerto cambiando lo mínimo posible

  • trocearArrayEnTrozos(["a", "b", "c", "d"], 2); // debe devolver [["a", "b"], ["c", "d"]].
  • trocearArrayEnTrozos([0, 1, 2, 3, 4, 5], 3); // debe devolver [[0, 1, 2], [3, 4, 5]].
  • trocearArrayEnTrozos([0, 1, 2, 3, 4, 5], 4); // debe devolver [[0, 1, 2, 3], [4, 5]].
  • trocearArrayEnTrozos([0, 1, 2, 3, 4, 5, 6], 3); // debe devolver [[0, 1, 2], [3, 4, 5], [6]].
  • trocearArrayEnTrozos([0, 1, 2, 3, 4, 5, 6, 7, 8], 4); // debe devolver [[0, 1, 2, 3], [4, 5, 6, 7], [8]].

Para probar la funcion se generará aleatoriamente un array de entre 8 y 15 posiciones con valores, a su vez, aleatorios entre 1 y N, con N > 1.Se mostrará la informacion en el navegador: la matriz generada, el producto de los pares e impares y la indicación de si son iguales o no.
